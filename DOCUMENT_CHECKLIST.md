# REQUIRED DOCUMENTS CHECKLIST

## REQUIRED FOR ALL APPLICATIONS

### Organization Docs
- [ ] Board resolution authorizing grant submission
- [ ] Proof of 501(c)(3) status (if applicable)
- [ ] Organizational chart
- [ ] Financial statements (most recent year)
- [ ] Audit report (if required/more than $750K federal funds)

### Grant Forms
- [ ] SF-424 (Application for Federal Assistance)
- [ ] SF-424A (Budget Information - Non-Construction)
- [ ] SF-424B (Assurances - Non-Construction)
- [ ] SF-424C (Budget Information - Construction) - if construction
- [ ] SF-424D (Assurances - Construction) - if construction
- [ ] Lobbying Disclosure (SF-LLL) - if applicable

### Narrative Sections
- [ ] Executive Summary
- [ ] Need Statement
- [ ] Project Description
- [ ] Methodology/Approach
- [ ] Evaluation Plan
- [ ] Sustainability
- [ ] Budget Narrative

---

## CONDITIONAL REQUIREMENTS

### If Hiring Staff
- [ ] Job descriptions for new positions
- [ ] Resume of key personnel

### If Partners
- [ ] Letters of commitment from partners
- [ ] MOUs or Letters of Agreement

### If Match Required
- [ ] Match commitment letters
- [ ] Documentation of match source

### If Construction/Rehabilitation
- [ ] Site control (deed, lease)
- [ ] Environmental review
- [ ] Architectural plans
- [ ] Cost estimates

### If Research
- [ ] IRB approval (if human subjects)
- [ ] Research methodology

---

## STATE/LOCAL GRANTS

Often require:
- [ ] State registration/compliance
- [ ] Local government endorsement
- [ ] Public hearing documentation
- [ ] Cost allocation plan

---

## UPLOAD INSTRUCTIONS

1. Create folder named: [Organization Name] - [Grant Name]
2. Upload all documents to that folder
3. Mark items as uploaded in checklist
4. Note any items not applicable

---

## COMMON REASONS GRANTS ARE REJECTED

- Missing board resolution
- Budget doesn't match narrative
- No 501(c)(3) status (for federal grants)
- Missing audit (if required)
- Incomplete SF-424
- Letters of support not on funder letterhead
- Missing match documentation
- Late submission

Double-check these before submitting!
